This package includes Apple-I compatible platform.
	6502 compatible CPU m65,
	Woz monitor binary image woz.mem
	simulation platform to run Woz monitro.
		in_data.mem is user input data stream (keyboard input)
		ap1.nsl is the top level logic for simulation.
	Also it has sample make scripts to layout it with Alliance

Steve Wozniak made his monitor for Apple-I in public domain.
We can it freely without problems against Apple computer.
